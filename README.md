
# First Python Projects 🐍

This repository contains my first Python programs, created while learning the basics of programming and problem-solving.

## Programs Included

- **hello.py**  
  A simple user-input greeting program.

- **calculator.py**  
  A menu-driven calculator that performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with divide-by-zero handling)

## How to Run

1. Make sure Python is installed on your system
2. Clone this repository or download the files
3. Open a terminal in the project folder
4. Run any program using:

   python filename.py

   Example:
   python calculator.py
